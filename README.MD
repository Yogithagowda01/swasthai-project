# SwasthAI – AI Health Assistant

## 📌 Description
SwasthAI is an AI-based health assistant that predicts diseases based on symptoms, asks basic questions like a doctor, and provides doctor availability, medicines, and alerts.

## 🚀 Features
- Symptom-based disease prediction
- Doctor-like questioning (duration, severity)
- Risk level detection
- Doctor recommendation with availability
- Basic medicine suggestions
- Alert system

## 🛠️ Tech Used
- Python
- Databricks (Spark + Delta Lake)
- AI Logic (Rule-based)

## ▶️ How to Run
1. Run the Python file
2. Enter symptoms
3. Answer questions
4. Get prediction and suggestions
